Toast Notification System

Click the buttons below to see different toast notifications in action

Usage Examples:

// Success notification
toast.success('Product added to cart!');

// Error notification
toast.error('Failed to process request');

// Warning notification
toast.warning('Only 3 items left!');

// Info notification
toast.info('Free shipping available');

// Custom duration (5 seconds)
toast.success('Message', null, 5000);

// Custom title
toast.success('Item saved', 'Success');

Features

  • ✓ Smooth slide-in animations
  • ✓ Auto-dismiss after 3 seconds
  • ✓ Manual close button
  • ✓ Animated progress bar
  • ✓ Stacks multiple toasts
  • ✓ Mobile responsive
  • ✓ Matches futuristic theme
